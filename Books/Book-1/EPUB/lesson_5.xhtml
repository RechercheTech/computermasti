Using a Keyboard

   In this lesson you will learn how to use a keyboard.

Moz: Today we will play some games. First tell me, what do you use to write in your book?
Jyoti: We use a pencil or a pen. Our teacher uses a chalk to write on the board.
Tejas: I have seen my sister use a keyboard to write on the computer.

Moz: The Keyboard is an important part of a computer. What do you see on it ?
Tejas: It has many small buttons. Some of them have  alphabets and numbers written on them. There are also some words like ‘Enter‛ written on them.
Jyoti: The keyboard is connected by a wire to the CPU of the computer.
Moz: Right. We can use a keyboard to give instructions to the computer The buttons on the keyboard are called keys.

Alphabet Keys

The keys on which you see alphabets: A – Z are called alphabet keys. These are used to write words.
