Tejas : I see ‘Tools‛ written on the screen.
What does it mean?

Moz : When you draw on a paper, you use pencil, brushes, colours and eraser. These are your drawing tools. In the same way, the tools for drawing using a computer can be found under Tools. A set of tools is called a Toolbar. The various  choices in any toolbar are called the Options.
